# ArmCare 机械臂远程运维管理平台 · 产品原型

## 项目简介

本项目基于自身在机械臂软件开发中的实际经验，针对机械臂远程运维场景设计的一套 Web 端管理平台原型(1920×1080)。
本原型为产品设计练习，非上线项目。所有图表数据均为模拟数据，仅用于展示交互逻辑与信息架构。

产品名称：ArmCare  
目标用户：工厂/实验室中的机械臂操作人员、设备维护工程师  
核心价值：在一个看板上完成状态监控、异常告警、历史数据追溯、运维工单管理，融合AI功能，提升故障响应效率。

## 版本说明
v1.0 实现传统运维管理功能
v2.0 在实现传统运维管理功能的基础上，新增智能运维助手

## 传统运维管理功能模块

| 模块 | 核心功能 |
| 实时监控看板 | 设备状态、关键指标、关节角度趋势图、最近告警滚动条 |
| 异常告警中心 | 告警列表、多维度筛选、告警详情、一键创建工单 |
| 历史数据查询 | 按设备/时间/数据类型查询并导出历史数据，趋势图+表格 |
| 远程运维工单 | 工单列表（卡片式）、新建工单、状态流转 |
| 设备管理 | 设备列表、编辑基本信息（IP、位置等） |

## 智能运维助手

### 交互位置

- **底部悬浮圆角对话框**，常驻于“异常告警中心”页面左下角，不干扰表格操作。

### 核心能力

运维人员在告警列表中点击任意一条告警，告警内容会自动填入助手输入框；点击发送后，助手根据预设规则模拟大模型返回**历史解决方案建议**。

### 预设回复逻辑（示例）

| 告警关键词 | AI 建议 |
| :--- | :--- |
| 关节1温度超过 | 检查散热风扇、清理滤网、降低负载。参考工单 #1023。 |
| 末端位姿偏离 | 重新校准位姿零点、检查夹爪松动。参考工单 #1087。 |
| 通信超时 | 检查线缆连接、重启通信模块、排查网络配置。参考工单 #1052。 |
| 电流异常 | 检查电机驱动模块、观察电流曲线。参考工单 #1103。 |
| 其他 | 检索历史案例库，建议查看错误代码。 |
> 当前为前端模拟逻辑，真实场景可接入企业知识库 + 大模型 API。
